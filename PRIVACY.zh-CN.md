# 青鸟 Qingniao 隐私政策

[English](PRIVACY.md) | **简体中文**

最后更新：2026-07-06

青鸟 Qingniao 是一款本地优先的 macOS 效率工具。其设计理念是：剪贴板历史、截图、设置与使用数据默认保留在你的 Mac 上。

## 本地存储的数据

青鸟可能会在本地存储以下数据：

- 剪贴板记录：文本、富文本的纯文本摘要、图像资源以及 Finder 文件引用。
- 剪贴板资源：位于应用支持目录中的图像原图、缩略图，以及富文本 RTF/HTML 资源。
- 设置：快捷键、开机启动偏好、搜索源显示设置、剪贴板保留期、截图保存目录、外观以及语言偏好。
- 搜索元数据：用于结果排序的本地黑名单条目与应用 / 命令使用次数。
- 截图文件：保存在你所选目录（默认 `~/Desktop`）中的 PNG 截图。

所有应用数据存储于 `~/Library/Application Support/Qingniao/`。

## 剪贴板历史

引导流程确认后剪贴板历史默认开启。你可以在应用中暂停剪贴板记录、调整保留期、删除单条记录，或清空全部剪贴板历史。

文件类型的剪贴板历史仅存储文件引用与路径，不会将文件内容复制进应用数据库。

## 截图

截图与标注均在本地处理。复制截图会将其写入系统剪贴板，并可能加入本地剪贴板历史。保存截图会将 PNG 文件写入你配置的截图目录（默认 `~/Desktop`），不会另建独立的截图历史。

## 权限

- **屏幕录制**：截图捕获所必需。
- **辅助功能**：按需申请 —— 仅在真正需要该权限的功能被使用时才申请，而非在引导流程中申请。
- **Apple Events / 自动化**：仅用于用户显式触发的白名单内置命令。
- **剪贴板访问**：在引导流程中进行说明，因为 macOS 不会为剪贴板读取单独弹出权限提示。

## 网络与上传

青鸟不会自动上传剪贴板历史、截图、文件、设置、搜索使用数据或诊断信息。不会将数据同步到云端服务，也不会使用你的数据训练模型。

应用的 **检查更新** 操作会在浏览器中打开 GitHub Releases 页面以便手动下载。青鸟不会自动下载、自动安装或重启以完成更新。

## 反馈邮件

反馈仅由用户主动发送邮件。在打开邮件应用之前，青鸟会先展示数据范围并允许你取消。生成的邮件可能包含：

- 应用版本与构建号。
- macOS 版本。
- 可选的错误摘要。
- 你自己填写的备注。

生成的邮件不会附带剪贴板历史、截图、文件或自动崩溃日志。

## 账号、支付与订阅

青鸟不包含账号、登录、注册、支付、订阅、许可证密钥或 Mac App Store 购买流程。

## 联系

如有隐私相关问题或反馈，请发送邮件至 [feedback@qingniao.app](mailto:feedback@qingniao.app)。
